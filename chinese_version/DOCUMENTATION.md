# 量子启发五声调式旋律生成 - 项目文档

本文件包含量子启发五声调式旋律生成项目的综合文档。

## 项目概述

本项目实现了一种新颖的方法，使用量子计算原理和新里曼音乐理论来生成中国五声调式旋律。

## 核心特性

- **量子启发算法**：使用量子游走和Grover算法进行旋律生成
- **新里曼理论适配**：将西方音乐理论概念适配到中国五声音阶
- **高有效性**：实现97.6%的有效五声音阶状态
- **双语支持**：提供中英文版本

## 核心组件

### 1. WPIM.py (加权音高交互模型)
- 主要旋律生成算法
- 实现带Grover约束的量子游走
- 双级约束机制确保音高有效性和流畅度

### 2. BMA.py (基础旋律算法)
- 简化的旋律生成方法
- 对比实验的基线

### 3. RuleBased.py
- 传统基于规则的旋律生成
- 经典方法对比

### 4. OpenQASM.py
- 量子电路实现
- 量子硬件执行的OpenQASM代码

### 5. IBM.ipynb
- IBM量子体验Jupyter笔记本
- 交互式量子电路测试

## 算法详情

### 量子游走过程
1. 使用五声音阶音符初始化量子态
2. 应用量子游走算子进行音符转换
3. 使用Grover算法进行振幅放大
4. 测量获得旋律序列

### 双级约束
- **第一级**：确保五声音阶有效性
- **第二级**：优化旋律流畅性和平滑性

### 新里曼理论适配
- 为五声音阶适配R/L/P操作
- 实现最小移动原则
- 创建五元环形音高网络

## 安装和使用

### 系统要求
- Python 3.8+
- Qiskit 2.0+
- IBM量子账户（用于硬件执行）

### 设置步骤
```bash
# 创建虚拟环境
python3 -m venv qiskit-env
source qiskit-env/bin/activate  # Linux/macOS
# 或 qiskit-env\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

### 运行代码
```bash
# 主旋律生成
python WPIM.py

# 四模型三指标对比实验
python comparison_experiment.py

# 单独算法
python BMA.py
python RuleBased.py

# IBM量子体验
jupyter notebook IBM.ipynb
```

## 实验结果

### 性能指标
- **有效五声音阶状态**：97.6%
- **电路深度**：261个门
- **量子比特数**：3个量子比特
- **采样次数**：1024

### 四模型对比实验
- **GT (Ground Truth)**：283个真实五声旋律基准
- **WPIM (Ours)**：加权相位干涉模型，最接近真实旋律
- **Native BMA**：原生量子基线模型
- **Rule-based**：传统硬规则方法

### 评估指标
- **MCS**：调式重心稳定度（宫音+徵音占比）
- **MPN**：基于W_base的过渡相似度
- **LSC**：长程连贯度（平均音程大小/12）
