# 面向五声调式的新里曼理论驱动旋律生成模型：量子启发的结构化概率方法

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue.svg)](https://www.python.org/)
[![Qiskit](https://img.shields.io/badge/Qiskit-2.0%2B-blueviolet.svg)](https://qiskit.org/)

本项目是论文 **《面向五声调式的新里曼理论驱动旋律生成模型：一种量子启发的结构化概率方法》** 的完整可复现代码实现。

## 项目简介

本文提出了一套完全以中国传统音乐理论为核心驱动的五声调式旋律生成模型：

- **理论基础**：将新里曼音乐理论适配到中国五声调式创作体系，构建了五元环形音高网络，定义了P/E/F型平滑进行规则
- **核心创新**：量子启发的双级递进约束机制，实现"全局锁定音高 + 局部控制流畅度"，解决传统方法"约束太死模板化，约束太松易跑调"的痛点
- **实验结果**：本开源实现实际大规模测试得到合法五声音阶态占比 **97.6%**，在随机采样波动范围内非常接近论文结果，算法正确性得到验证。专业音乐从业者双盲听评显著优于传统方法

## 核心思想

 1. **五元环形音高网络**：把宫商角徵羽五个正音转化为拓扑图中的节点，合法旋律进行转化为节点连接
 2. **逐音符生成流程**：采用逐音符迭代生成方式，每一步都基于当前位置通过量子游走预测下一个音符
 3. **双级Grover约束机制**：第一级Grover约束锁定五声音阶合法性，第二级Grover约束优化旋律进行流畅性，实现"全局锁定音高 + 局部控制流畅度"
 4. **量子启发概率调整**：借鉴Grover搜索算法的振幅偏置思想，给符合五声规范的旋律走向赋予更高概率
 5. **结构化旋律游走**：模拟真人作曲家"多方向并行构思 → 按规则筛选优化 → 确定最终选择"的创作过程

## 快速开始

### 1. 创建虚拟环境

```bash
python3 -m venv qiskit-env
source qiskit-env/bin/activate  # Linux/macOS
# 或: qiskit-env\Scripts\activate  # Windows
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

 ### 3. 运行主程序生成旋律

```bash
python pentatonic_melody_simple.py
```

程序将输出：
- 统计结果（合法态占比等）
- 生成MIDI文件 `pentatonic_quantum_melody.mid`
- 测量分布图 `measurement_histogram.png`

### 4. 运行对照实验

```bash
python comparison_experiment.py
```

对比三种方法：
1. 经典随机游走
2. 量子游走（无Grover约束）
3. 量子游走 + Grover约束（完整模型）

验证量子方法在提升合法五声音阶概率上的优势。

### 5. 测试不同Grover迭代次数

```bash
python grover_test.py
```

## 模型参数说明

| 参数 | 含义 | 默认值 |
|------|------|--------|
| `Kw` | 量子游走步数 | 16 |
| `Ko` | Grover迭代次数 | 1 |
| `shots` | 测量采样次数 | 1024 |
| `total_measures` | 生成旋律小节数 | 8 |
| `tempo` | MIDI速度(BPM) | 120 |

五声音阶映射（C调宫调式）：

| 节点 | 五声音级 | 简谱 | MIDI音高 | 二进制编码 |

| v0 | 宫 | 1 | 60 (C4) | 000 |
| v1 | 商 | 2 | 62 (D4) | 001 |
| v2 | 角 | 3 | 64 (E4) | 010 |
 | v3 | 徵 | 5 | 67 (G4) | 011 |
 | v4 | 羽 | 6 | 69 (A4) | 100 |
